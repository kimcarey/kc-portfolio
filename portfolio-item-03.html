Color Picker

Know your RGB?

This is a fun game where the user is shown a RGB value and 6 different color circles. The user must choose the correct match that corresponds to the given RGB value.

Full Code Here

Tech Stack: JavaScript, HTML, CSS

Process: The first thing to do was add color to the buttons. I tested it out manually using the getElementByClassName() and setAttribute() functions before attempting changing the color dynamically. Once I had this working, I was ready to get to work.

To make the game work, I needed to be able to select random colors between 0 and 255 (the total color values between R, B, and G), using Math.round() and Math.random() and setting that formula to red, green, and blue variables. You can then apply a for loop in order to change the color of the buttons.

Then, I had to work on picking a random button as the winning answer and making it so that, when the player clicks on the correct button, they see a message that they've won. I added an event listener in order to know which button the player selected. The last piece was resetting the game, so that it will clear the previous round and start a new game.
