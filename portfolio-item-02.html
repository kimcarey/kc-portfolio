Ship-wrecked!

An Adventure Game

This was one of the final project for Udacity's Python course. The goal was to create a simple version of an old-fashioned text-based adventure game.

Tech Stack: Python

Process: Once I had an idea for the plot line, the first step was to compose descriptive messages to set up the game. It wouldn't be a "choose your own" adventure without the element of choice. I came up with a couple options for the player to choose from based on what environment they are in within the game -- in this case, beach and the forest (do you hide in the cave or approach the other inhabitants?). Of course, the game also needed an element of randomness and I chose to apply that to the cave scenario. I can't share what it is or else that would spoil the fun!

The biggest challenge to solve for this game was ensuring that the player enters a valid input. The game will keep asking for a valid answer before progressing. Once happy with the flow of the game, I refactored my code to keep it clean.
